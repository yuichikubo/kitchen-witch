🌿 Decap CMS セットアップ手順

台所の魔法 Kitchen Witch — /admin から管理画面にログインするまでの全手順（所要時間: 約10分）

この手順を上から順に実行するだけで、https://kitchen-witch-magic.netlify.app/admin/ から管理画面にログインできるようになります。

1
GitHub

ZIPを解凍して、ファイルをGitHubにアップロードする

  1. ZIPファイルを解凍する（フォルダ名は何でも可）
  2. GitHubで kitchen-witch-magic のリポジトリを開く
  3. 「Add file」→「Upload files」をクリック
  4. 解凍したフォルダの中身をすべてドラッグ＆ドロップ
    ※フォルダごとではなく、中のファイル・フォルダをすべて選択してください
  5. コミットメッセージ欄に「add Decap CMS」などと入力
  6. 「Commit directly to the main branch」→「Commit changes」をクリック
既存ファイルと同名（index.htmlなど）がある場合は上書きされます。管理画面スクリプト入り版に差し替えてください。

2
Netlify

Netlify Identity を有効にする

  1. app.netlify.com を開いてサイトを選択
  2. 上のタブ 「Site configuration」 をクリック
  3. 左メニュー 「Identity」 をクリック
  4. 「Enable Identity」 ボタンをクリック
Identity が有効になると設定項目が表示されます。
このステップを飛ばすと管理画面にログインできません。必ず先に実行してください。
3
Netlify

Git Gateway を有効にする（CMSがGitHubに書き込めるようにする）

  1. 「Site configuration」→「Identity」ページを開く
  2. ページ下部の 「Services」 セクションを見つける
  3. 「Git Gateway」の 「Enable Git Gateway」 をクリック
これで管理画面から「保存」すると自動でGitHubにコミットされます。
4
Netlify

ログイン用のアカウントを作る（管理者招待）

  1. 「Site configuration」→「Identity」→ 「Invite users」 をクリック
  2. 管理者のメールアドレスを入力して「Send」
  3. 届いた招待メールの 「Accept the invite」 をクリック
  4. パスワードを設定して完了
招待メールが届かない場合は迷惑メールフォルダを確認してください。

補足: Netlify Identityは無料プランで最大5ユーザーまで登録できます。

5
完了確認

管理画面にアクセスしてログインする

  1. ブラウザで下記URLを開く
https://kitchen-witch-magic.netlify.app/admin/
  1. メールアドレスとパスワードを入力してログイン
  2. ダッシュボードが表示されれば設定完了 🎉
スマホ・タブレットからも同じURLでアクセスできます。

管理画面でできること

  • 🏠 トップページ — キャッチコピー・お客様の声・季節レシピカードを編集
  • 🌿 コンセプト — 本文・6原則・四季セクションを編集
  • 💼 サービス — 料金・特徴リスト・FAQを追加・編集・削除
  • 🍲 レシピ記事 — 新規作成・Markdown本文・画像・季節タグで管理
  • ⚙️ サイト設定 — SNS URL・コピーライト・OGP画像
現在の構成について: 管理画面での保存はJSONファイルとしてGitHubに書き込まれます。現在の静的HTMLに自動反映させるには、11tyやAstroなどのSSG（静的サイトジェネレーター）との連携が必要です。まずはレシピ記事の蓄積から始めて、後からビルドシステムを追加する形を推奨します。
